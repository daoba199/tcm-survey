🏥 中医体质自测问卷

填写说明：请根据您最近一个月的实际感受，勾选符合的选项。提交后可查看体质分析结果。
一、脾胃虚弱自测 0
符合3条以上建议调理脾胃
吃一点就饱、容易腹胀、不消化
大便不成形、黏马桶、冲不干净
面色发黄、暗沉、没光泽
容易累、懒得动、说话没力气
肌肉松垮、虚胖、喝水都长肉
舌头胖大、有齿痕、舌苔偏白
二、体内湿气自测 0
符合3条以上建议祛湿
身体沉重、犯困、总睡不醒
头脸爱出油、长痘、头发油得快
白带多、黏腻、有异味（女性）
下肢浮肿、早上眼皮肿
情绪闷、烦躁、提不起劲
舌苔厚腻、发白或发黄
三、气血不足自测 0
符合3条以上建议补气血
脸色苍白/萎黄、嘴唇指甲色淡
头晕、蹲起发黑、心慌
月经量少、推迟、颜色淡
掉发多、干枯、白发早生
失眠多梦、容易惊醒
手脚常年冰凉、怕冷
四、睡眠质量自测 0
符合3条以上可能存在脾虚湿重+气血不足型失眠
入睡困难，躺床上半小时以上还睡不着
梦特别多、乱梦纷纭，像演电视剧
容易醒，夜里醒1～3次，醒了难再睡
早上醒得早，凌晨3～5点醒
睡醒还是累，越睡越困、没精神
心烦、心慌、容易紧张、焦虑
头顶闷、头胀、记忆力下降
口苦、口干、夜里想喝水
女性伴随月经不调、量少、情绪暴躁
舌头偏红/偏淡，舌苔厚腻
